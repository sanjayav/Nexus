# Aeiforo Sustainability Reporting Dashboard

## Project Overview
React + TypeScript SPA for ESG/sustainability reporting. Built with Vite, Tailwind CSS, React Router v7, Recharts. Deployed on Vercel.

## Tech Stack
- **Framework**: React 18 + TypeScript
- **Bundler**: Vite 5
- **Styling**: Tailwind CSS 3
- **Routing**: React Router DOM v7
- **Charts**: Recharts
- **Icons**: Lucide React
- **Deployment**: Vercel (SPA with client-side routing)

## Commands
- `npm run dev` — Start dev server
- `npm run build` — TypeScript check + Vite build
- `npm run preview` — Preview production build

## Architecture
- `src/App.tsx` — Root router
- `src/components/AppShell.tsx` — Main layout shell
- `src/auth/AuthContext.tsx` — Auth context/provider
- `src/pages/` — Route pages (Dashboard, Reporting, CarbonAccounting, etc.)
- `src/data/` — Static data (moduleData, pttgcData)

## Conventions
- Glass UI / Nexus design system aesthetic
- Tailwind utility classes (no CSS modules)
- Functional components with hooks
- Demo auth with hardcoded credentials (Marklytics domain)
